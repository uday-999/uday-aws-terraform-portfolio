# Day 07 Notes

## Basic Types

string → text values

number → numeric values

bool → true/false

## Collection Types

list(string) → ordered values

set(number) → unique unordered values

map(string) → key-value pairs

## Complex Types

object → structured data

Example:

instance_type
volume_size

## Validation

Validation prevents invalid input.

Example:

Only allow:

dev
staging
prod

## Biggest Learning

Terraform type constraints make infrastructure safer, cleaner, and more predictable.
